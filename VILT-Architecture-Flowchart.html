Live Sessions (VILT) Architecture

SkillUp LMS / Based on meeting April 10, 2026 / Harpreet, Govind, Navdeep, Rupali, Nelson

Core Principle
≡

Separate Course Structure from Delivery Method

A course is a package of content. VILT is not a course type. It is a mode of delivery. The course structure (Program > Course > Module > Lesson) stays the same regardless of how it is delivered.

Video Reading Lab Live session Quiz
Three Modes of Delivery
📖

Self-Paced

No instructor. No deadlines. Learner consumes content on their own schedule. Videos, readings, quizzes, labs.
No timeline pressure
🎤

VILT (Instructor-Led)

Heavily instructor-driven. Still includes quizzes, projects, reading material. Think university: lectures + homework.
Date + Time + Presenter required
🔄

Blended

Mix of self-paced and live sessions. Most common model. Weekday self-study, weekend VILT sessions.
Start/End dates visible

Learner View

📋 Course Structure (Linear View)

  • All modules and lessons listed in order
  • Visual indicator per item: badge, icon, or color showing delivery mode
  • VILT items show date/time of session inline
  • Completed VILT shows "Done" status
  • If recording available, show recording link

📅 Calendar Tab (Course-Level)

  • Tab alongside course structure
  • Shows only live sessions for this specific course
  • Session topic, date, time, presenter
  • Correlates with course structure

🌍 Global Calendar

  • Across all enrolled courses and programs
  • Aggregated view of all upcoming live sessions
  • Learner sees full schedule at a glance

🔔 Nudges and Reminders

  • "Session on Module 3 in 3 days. Complete Modules 1 and 2."
  • Upcoming session topics and recommended pre-reading
  • No blocking: learner can still attend even if behind on self-paced
Key Rules
🚫
No Duplicate Courses

Never create separate VILT and self-paced versions of the same course. One course, one instance.

🔓
No Locking VILT Behind Prerequisites

Learner can attend a live session even if they haven't finished prior self-paced modules. Nudge, don't block.

🎓
"Pure VILT" Still Has Supporting Content

Even instructor-led courses have quizzes, projects, readings. Like university: lectures + homework. Not "blended" for having a textbook.

⏰
VILT = Time-Bound, Self-Paced = Open

Self-paced shows no deadlines. Any VILT component adds start/end dates and schedule pressure to the learner view.
