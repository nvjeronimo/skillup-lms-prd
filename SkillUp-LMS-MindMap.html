Identified Gaps

1. "Completed" States

No CTA defined for finished courses/programs. Missing: Review, View Certificate, Retake.

2. Empty States

Zero wireframes for empty states: first-time user, no results, no sessions, no recordings.

3. Sub-tabs Not Wireframed

Assessments, Projects, Progress and Certificates exist in program nav but are not designed.

4. Live Now vs Starting Soon

No clear distinction between a session that already started and one about to start.

5. Global Calendar

Tab exists but wireframe is incomplete for cross-course view with conflict detection.

6. Recording Failed

No error state for when recording processing fails.

7. Session Cancelled/Rescheduled

No wireframe for when instructor cancels or reschedules a session.

8. Expired/Overdue

No state for courses with expired deadlines or closed enrollment windows.

9. Mandatory vs Optional

No visual differentiation between mandatory (compliance) and optional courses.

10. Conflict Detection

No visual indicator for when two live sessions overlap.

11. Offline/Download

Offline access to self-paced content. Download recordings.

12. Re-enrollment

Retaking a completed course. Progress reset vs additive?

13. Cohort Context

Group learning with shared timeline and progress visibility.

14. Notifications

Alert system: session starting, recording available, certificate, deadline, cancellations.
