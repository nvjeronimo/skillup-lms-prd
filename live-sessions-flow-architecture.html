Module Reference Patterns
After Module 4: Predictive Models Session follows a specific module
Covers Courses 1-2 Program-level session spanning multiple courses
After Module 3 . Before Final Project Session between two specific points
Kickoff Session Session before any module content
After Capstone Submission Final wrap-up session

Open question from meeting: Progress calculation for blended programs.
If a program is 4 courses + 5 live sessions, do sessions count toward overall progress? Options: (A) Sessions count as completion items. (B) Sessions are parallel and don't affect the progress bar. (C) Attendance is tracked separately as a KPI (current V4 shows "2/5 Live lessons attended"). Recommendation: Option C — keep progress tied to self-paced content, track attendance as a separate KPI.
